## 練

[練](http://kotobank.jp/word/練)で縛るかも


- [訓練](http://kotobank.jp/word/訓練)  
  あることを教え、継続的に練習させ、体得させること。「きびしい―にたえる」「―して生徒を鍛える」
- [試練](http://kotobank.jp/word/試練)  
  信仰・決心のかたさや実力などを厳しくためすこと。また、その時に受ける苦難。「厳しい―を受ける」「―に耐える」
